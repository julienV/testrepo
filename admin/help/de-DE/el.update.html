Update Überprüfung

Tip
Prüfen Sie regelmäßig ob Updates verfügbar sind. Regelmäßiges aktuallisieren sorgt für Sicherheit und und rüstet Ihre Seite mit neuen nützlichen Funktionen aus.

Falls Sie die Heimat von EventList nicht regelmäßig besuchen oder die Updatebenachrichtigung nicht nutzen, sollten Sie in regelmäßigen Abständen den Button "Updatecheck" anklicken um auf eine neue Version von EventList zu prüfen.

Hier wird eine XML-Datei auf schlu.net ausgelesen welche aktuelle Daten enthält. Diese werden danach im Popup angezeigt inklusive einer Angabe ob Sie die aktuelle Version von EventList einsetzen oder nicht.
