Housekeeping

Tip
Remove unnecessary images regularly.

Here you can clean up the image database with a simple click on the appropriate icon. The clean up pulls all used images out of the database and then compares the result with all available images. Not or not longer used images are deleted in this process without further inquiry. Note that this feature should be used only if the Event List images are not used elsewhere, in articles, etc. Should this be the case, you can also use the Joomla! Media Manager or the EventList image selection to selectively delete individual images.

This feature is available both for venues and events.
