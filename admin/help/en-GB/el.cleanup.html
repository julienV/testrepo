Housekeeping

Tip
Remove unnecessary images regularly.

Here you can clean up the images in your database with a simple click of the appropriate icon. This action extracts all used images from the database and then compares the result with all available images. Images which are not used (or no longer used) are deleted immediately.

Note that this feature should be used only if the EventList images are not used elsewhere (such as content items in other articles). If you are using your EventList images in other areas, you can also use the Joomla! Media Manager or the EventList image selection to selectively delete individual images.

This feature is available both for venues and events.
