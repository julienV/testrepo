Edit Venue

Description

Edit your venue items or add new ones from that screen.



Tip
Use Plugins to add additional functionality to EventList. You can connect your events for example with a reservationsystem like "akreservations' or Galleries with the help of Plugins.

Navigation

To access this screen use: Components -> EventList -> Add Venue or by clicking a venue in the overview

User Access Level

Super Administrator Administrator

Toolbar Icons

Apply: Saves the venue and returns to the current view.

Save: Saves the venue and returns to the overview.

Cancel: Cancels the action. Any modification is lost. If an item is opened to edit, the venue Item remains checked-in.

Help: Opens this help screen.


Main publishing area

Venue: Enter or edit (if needed) the title of the venue here.

Published: Select Yes to publish the venue or No to unpublish. Published venues are displayed on the Frontend and you can assign events to them.

Textfield: This is for the Venue description. You can use Plugins in this field.

Advanced publishing area

Address: Type here the street, city, zip and the country code. These informations are required if you use a map service.

Image: Choose an image for this venue (flyer) or upload it.

Metadata information: Enter or edit (if needed) the Meta Description and the Meta Keywords of the venue here.
